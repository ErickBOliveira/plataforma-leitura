\# Plataforma Ler \& Aprender



Projeto desenvolvido para a Atividade Extensionista do curso de CST em Análise e Desenvolvimento de Sistemas – UNINTER.



\## Objetivo



Desenvolver uma plataforma digital para incentivo à leitura, compreensão textual e inclusão digital de alunos do Ensino Fundamental, utilizando recursos tecnológicos para apoiar o processo de ensino-aprendizagem.



\## Tecnologias Utilizadas



\* HTML5

\* CSS3

\* JavaScript

\* Firebase Authentication

\* Cloud Firestore

\* Firebase Hosting



\## Funcionalidades



\* Cadastro de Professores

\* Cadastro de Turmas

\* Cadastro de Alunos

\* Leituras Digitais

\* Atividades de Compreensão Textual

\* Sistema de Gamificação e Ranking

\* Painel do Professor

\* Painel do Aluno

\* Controle de Desempenho dos Estudantes



\## Aplicação Prática



O projeto foi aplicado na:



\*\*Escola de Referência em Ensino Fundamental Professora Maria Bernadete Marins de Brito\*\*



Salgueiro – Pernambuco



Com acompanhamento da professora \*\*Joelma Rufino\*\* e participação de alunos do \*\*6º Ano do Ensino Fundamental\*\*.



\## Objetivos de Desenvolvimento Sustentável (ODS)



O projeto contribui diretamente para:



\* ODS 4 – Educação de Qualidade

\* ODS 9 – Indústria, Inovação e Infraestrutura
